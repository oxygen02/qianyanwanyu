# 2026-05-19 工作记录

## 落墨页设置面板重构

### 问题
- 设置面板（功能按键区域）被 Canvas 遮挡，浮于输入区下方无法看到
- 所有数值设置项使用加减按钮（cv-stepper），交互不直观
- 选择设置时 Canvas 实时预览已实现（onChange 已绑定 _triggerRender）

### 解决方案

**1. 层级修复（浮于 Canvas 上方）**
- 将设置面板从 `cover-view` 改回普通 `view`，z-index 提升至 10000
- font-panel z-index: 10001，template-panel z-index: 10002
- 面板在 `page-root` 外独立放置，fixed 定位

**2. 所有加减按钮改为 slider 滑动条**
- WXML：设置面板重写，使用新 `sp-` 前缀组件类
- 涉及所有数值型设置：字号、浓度、上/下/左/右边距、老旧感、纹理、阴影、光照、行距、字距、墨迹变化、浸染、套印、破损
- 每个 slider 右侧显示当前数值

**3. 实时预览**
- slider `bindchange` 绑定已有的 `onXxxChange` 函数
- `onXxxChange` 均调用 `_triggerRender()`，实现实时预览

### 修改的文件
- `miniprogram/pages/luomo/luomo.wxml`：完整重写设置面板，改为普通 view + slider
- `miniprogram/pages/luomo/luomo.wxss`：废弃 cv-* 样式，新增 sp-* 样式系统 + slider 样式
- JS 无需修改（onChange 函数已兼容 slider 的 e.detail.value）
