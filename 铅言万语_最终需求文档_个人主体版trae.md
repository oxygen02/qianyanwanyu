## 一、产品定位

### 1.1 一句话定义
将文字转化为具有**油墨浸染质感、岁月纸张纹理、铅字印刷美感**的仿真书页。

### 1.2 核心价值
- **情感**：让文字拥有"被阅读"的仪式感
- **审美**：中式克制美学 + 现代横排为主（70%+）
- **实用**：一键生成适合朋友圈/小红书/抖音传播的高品质图文
- **自由**：所有模板、字体、纸张**完全免费**

### 1.3 目标用户
| 层级 | 年龄 | 场景 | 偏好模板 |
|------|------|------|----------|
| 核心 | 18-35岁 | 社交分享、读书笔记、文案创作 | 现代散文、打字机体、手写日记 |
| 次要 | 35-50岁 | 读书笔记、金句收藏、传统文化 | 民国期刊、旧档案、牛皮信笺 |
| 边缘 | 50岁+ | 怀旧情感、古籍体验 | 线装古籍、宋刻本 |

### 1.4 命名
**铅言万语** —— 铅字之言，万千语汇。四字符合微信小程序审核要求。

## 四、核心功能

### 4.1 墨写（文字成书）

用户提供文字，系统实时渲染为仿真书页。每输入一个字符，画布即时更新。

**编辑器布局：**
```
┌──────────────────────────────────┐
│  状态栏: 第1页/共3页 | 286字 | ✅ 已自动保存  │
├──────────────────────────────────┤
│     ┌────────────────────┐       │
│     │   仿真书页画布      │       │  ← Canvas 2D (屏幕65%+)
│     │   实时渲染          │       │
│     └────────────────────┘       │
├──────────────────────────────────┤
│  [📄纸张] [🔤字体] [📐排版]      │  ← 样式快捷栏
│  [✨特效] [📝注释] [🔖印章] [💧水印] │
├──────────────────────────────────┤
│  ┌────────────────────────────┐  │
│  │ 在此输入或粘贴文字...      │  │  ← 底部输入框
│  └────────────────────────────┘  │
└──────────────────────────────────┘
```

**核心交互：**
- 点击书页空白处 → 唤起系统键盘 → 竖线游标定位
- 每输入一个字符 → Canvas增量重绘（防抖50ms）
- 文字渲染延迟 ≤ 100ms
- 支持拼音输入法候选词确认后才触发渲染
- 删除键回退即时响应

### 4.2 拾文（文章转书）

**双路径输入 → 统一清洗管道：**

```
┌─────────────────────────────────────────────────────────┐
│                    用户输入层                              │
│  ┌──────────────────┐      ┌──────────────────────┐       │
│  │  路径A：粘贴链接   │      │  路径B：手动复制粘贴    │       │
│  │  （网页/非公众号） │      │  （公众号首选方式）     │       │
│  └────────┬─────────┘      └──────────┬───────────┘       │
│           │                           │                    │
│           ▼                           ▼                    │
│  ┌──────────────────┐      ┌──────────────────────┐       │
│  │ 前端URL校验       │      │ textarea接收          │       │
│  │ 公众号链接→弹提示  │      │ 格式识别              │       │
│  └────────┬─────────┘      └──────────┬───────────┘       │
└───────────┼───────────────────────────┼────────────────────┘
            │                           │
            ▼                           ▼
┌───────────────────────────────────────────────────────────┐
│                 统一清洗管道（服务端云函数）                  │
│  第一层：DOM级清洗（@mozilla/readability）                   │
│    ├─ HTML → JSDOM解析                                      │
│    ├─ Readability.parse() → 提取正文候选区                   │
│    ├─ 返回置信度分数 (contentScore)                          │
│    └─ 输出：{ title, textContent, html, score }             │
│  第二层：自定义规则层（Post-Processing Pipeline）             │
│    ├─ 关键词黑名单过滤（引导关注/扫码/广告语）                │
│    ├─ 短段落过滤（<15字符且不含标点的孤立行）                │
│    ├─ 重复内容去重（连续相同段落合并）                       │
│    ├─ 超链接剥离（只留文字，去掉<a>标签地址）                 │
│    ├─ 图片筛选（去除≤50x50的小图标/追踪像素）               │
│    └─ 特殊符号清理（HTML实体/多余空白/零宽字符）              │
│  第三层：置信度分级 & 分发                                   │
│    ├─ score ≥ 80：高置信 → 直接进入预览                      │
│    ├─ score 50-79：中置信 → 预览 + 标注"可能不完整请检查"     │
│    └─ score < 50：低置信 → 提示切换为手动粘贴模式            │
└───────────────────────────┬───────────────────────────────┘
                            │
                            ▼
┌───────────────────────────────────────────────────────────┐
│                   预览 & 编辑确认层（前端）                     │
│  ├─ 标题 + 作者 + 来源                                     │
│  ├─ 前300字正文预览（可展开全文）                            │
│  ├─ 预估页数 / 总字数 / 图片数量                             │
│  ├─ 「确认转换」按钮 → 进入书页排版引擎                     │
│  ├─ 「编辑调整」按钮 → 可删段/可调序/可补漏                 │
│  └─ 「重新提取 / 改为手动粘贴」备选入口                      │
└───────────────────────────────────────────────────────────┘
```

**公众号文章策略：**
- 检测到公众号链接 → 弹窗提示："微信文章建议直接复制文字粘贴，效果最好"
- 提供「仍要尝试」选项给高级用户
- 手动粘贴路径全面优化，自动过滤底部关注引导、文中广告

## 五、排版渲染引擎

### 5.1 渲染管线

```
用户输入文字 → 文本预处理(分段/分行/测量)
    ↓
纸张层渲染（底色 + Perlin噪声纹理 + 径向渐变 + 边缘效果）
    ↓
文字层渲染（逐字绘制 → 墨迹特效叠加 → 晕染扩散）
    ↓
装饰层渲染（版框/栏线/页眉页脚/页码）
    ↓
注释层渲染（脚注/眉批/书耳/边注）
    ↓
印章/签名层（拖拽定位 + Canvas绘制 + 做旧处理）
    ↓
水印层（半透明叠加）
    ↓
合成输出（离屏Canvas双缓冲 → 显示Canvas）
```

### 5.4 墨迹模拟算法（8种特效）

```javascript
class InkEffectRenderer {
  config = {
    inkVarRange: 15,        // 墨色变化范围 (RGB 0-255)
    flyWhiteProb: 0.003,    // 飞白概率 (0.3%)
    heavyInkProb: 0.003,    // 重墨概率 (0.3%)
    jitterRange: 0.3,       // 字符微抖动范围(px)
    chipSize: { min: 1, max: 2 },  // 铅字缺角大小(px)
    splashDensity: 0.003,   // 墨点溅射密度
    blurRadius: 0.5~1.5,    // 晕染模糊半径(px)
  }

  applyInkEffect(ctx, char, x, y) {
    // 1. 深浅随机
    const inkVar = (Math.random() - 0.5) * this.config.inkVarRange;
    ctx.fillStyle = this.adjustRGB(baseColor, inkVar);

    // 2. 飞白瑕疵 (0.3%概率)
    if (Math.random() < this.config.flyWhiteProb) {
      ctx.globalAlpha = 0.4;
      this.brightness(ctx, 40);
    }

    // 3. 重墨瑕疵 (0.3%概率)
    if (Math.random() < this.config.heavyInkProb) {
      this.darken(ctx, 30);
    }

    // 4. 字符微抖动
    const jitterX = (Math.random() - 0.5) * this.config.jitterRange * 2;
    const jitterY = (Math.random() - 0.5) * this.config.jitterRange * 2;

    // 5. 绘制字符
    ctx.fillText(char, x + jitterX, y + jitterY);

    // 6. 铅字缺角
    if (Math.random() < 0.05) {
      this.applyChipEffect(ctx, x, y, char);
    }

    // 7. 晕染扩散
    ctx.shadowBlur = this.config.blurRange;
    ctx.shadowColor = baseColor;
    ctx.fillText(char, x + jitterX, y + jitterY);

    // 8. 墨点溅射
    if (Math.random() < this.config.splashDensity) {
      const sx = x + Math.random() * charWidth;
      const sy = y + Math.random() * charHeight;
      ctx.fillStyle = baseColor;
      ctx.beginPath();
      ctx.arc(sx, sy, 1, 0, Math.PI * 2);
      ctx.fill();
    }
  }
}
```
